サービス概要

本サービスは文章データのデータセットを元に， 文章を分類するモデルを作成する無料サービスです． 誰でも簡単に作成することができます． 用意していただくものは，大量の文章データだけです． モデル一覧からモデル作成へ移動し以下の学習モデルの作り方を参考にしてください．

学習モデルの作り方

学習器の作成ページへ移動し，フォームを入力してください．

{% load static %}

ユーザIDはユーザを識別するのに使います．

パスワードはモデルの管理（編集，削除）に使います．

タイトルはモデルのタイトルです．モデル一覧にそのタイトルが表示されます．

説明はそのモデルの分類する分野や，ラベルを記述して下さい．


学習データは次の項目を満たす必要があります．

  • "text1", "class1"のような形式で，csvファイルであること．
  • データ数は分類するクラスごとに50個以上あること（100個以上あれば分類精度が上がる）．

    • データセットの拡張子は.csvです．データセットはサーバに保存されます


    最後にモデル作成というボタンを押してください． モデルの作成にはデータの数に比例して数時間かかります．．


    モデル一覧からモデルを選択すると詳細画面に移動します． その詳細画面にはモデルのハッシュ値や，サンプルで文章分類できるフォームがあります．

    APIの使いかた
